# Constructs Module

CDK constructs for various AWS services.

## Available Constructs

- [Base](base.md) - Base construct classes and environment mixins
- [Assets](assets.md) - Lambda and Docker image assets
- [Batch](batch.md) - AWS Batch infrastructure and job management
- [CloudWatch](cw.md) - Monitoring dashboards and alarms
- [DynamoDB](dynamodb.md) - DynamoDB tables and indexes
- [EC2](ec2.md) - EC2 instances, VPCs, and networking
- [EFS](efs.md) - Elastic File System and access points
- [S3](s3.md) - S3 buckets and policies
- [Service](service.md) - Complete service infrastructure
- [SSM](ssm.md) - Systems Manager parameters
- [Step Functions](sfn.md) - State machines and fragments
